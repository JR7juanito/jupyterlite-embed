## 🎯 Cómo embeber en tu sitio web

Copia el código iframe de la columna correspondiente en la tabla de arriba, o usa este formato general:

```html
<iframe 
  src="https://jr7juanito.github.io/jupyterlite-embed/lab/index.html?path=NOMBRE_NOTEBOOK.ipynb"
  width="100%" 
  height="800px"
  frameborder="0">
</iframe>
```

### Opciones de visualización

- **Vista Lab completa**: `/lab/index.html?path=NOTEBOOK.ipynb`
- **Vista Notebook clásica**: `/notebooks/index.html?path=NOTEBOOK.ipynb`
- **Consola Python**: `/repl/index.html`

## ⚡ Características

- ✅ **Sin instalación**: Ejecuta Python directamente en el navegador
- ✅ **Sin servidor**: Todo funciona con WebAssembly (Pyodide)
- ✅ **Modo oscuro**: Tema dark activado por defecto
- ✅ **Interactivo**: Edita y ejecuta el código en tiempo real
- ✅ **Privado**: Tu código no sale de tu navegador
- ✅ **Gratuito**: Hosted en GitHub Pages

---

## 🛠️ Desarrollo local

Si quieres probar localmente:

```bash
# Instalar JupyterLite
pip install jupyterlite-core jupyterlite-pyodide-kernel

# Construir el sitio
jupyter lite build --contents content --output-dir _output

# Servir localmente
jupyter lite serve --output-dir _output
```
