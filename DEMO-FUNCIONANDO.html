🚀 ¿Qué estás viendo?

Abajo verás consolas Python reales funcionando completamente en tu navegador. Estos embeds usan URLs públicas de JupyterLite, así que funcionan inmediatamente sin necesidad de desplegar nada.

💡 Cómo interactuar:

  • Escribe código Python en la consola (por ejemplo: print("Hola"))
  • Presiona Shift + Enter para ejecutar
  • Prueba operaciones: 2 + 2, import random; random.randint(1,10)
  • El código se ejecuta en tu navegador usando WebAssembly (Pyodide)
🎯 Próximo paso: Copia el código de cualquier iframe y úsalo en tu sitio web, CodePen, JSFiddle, o cualquier plataforma HTML. ¡Funciona en todas partes!

🎨 Personalización

Puedes personalizar los embeds con parámetros URL:

  • ?toolbar=1 - Mostrar barra de herramientas
  • ?toolbar=0 - Ocultar barra de herramientas
  • ?kernel=python - Seleccionar kernel automáticamente
  • ?theme=JupyterLab%20Dark - Tema oscuro
  • ?theme=JupyterLab%20Light - Tema claro

Ejemplo con múltiples parámetros:

https://jupyterlite.github.io/demo/repl/index.html?kernel=python&toolbar=1&theme=JupyterLab%20Dark

🚀 ¿Quieres tu propia versión?

Los embeds de arriba usan URLs públicas de JupyterLite. Si quieres desplegar tu propia versión con notebooks personalizados:

  1. Sube el proyecto a GitHub
  2. Activa GitHub Pages con GitHub Actions
  3. En 2-5 minutos tendrás tu URL: https://tuusuario.github.io/tu-repo/
  4. Usa esa URL en tus embeds para mostrar tus notebooks personalizados
