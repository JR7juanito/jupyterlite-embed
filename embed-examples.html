🚀 JupyterLite Embed - Ejemplos

Copia y pega estos códigos para embeber Jupyter en tu sitio web

📋 Instrucciones de Uso

  • Cada ejemplo muestra un iframe diferente de JupyterLite
  • Haz clic en "Copiar Código" para copiar el iframe al portapapeles
  • Pega en tu HTML, blog, o documentación
  • Reemplaza <TU-URL> con tu URL real de despliegue
  • Prueba en CodePen, JSFiddle, o tu sitio web

💻 Opción 1: Consola Python Interactiva (REPL)

Una consola Python simple y ligera. Perfecta para documentación o tutoriales rápidos. Los usuarios pueden escribir y ejecutar código Python directamente.

⚠️ Reemplaza esto: <TU-URL> por tu URL de GitHub Pages, Vercel, o Netlify
Ejemplo: https://tuusuario.github.io/jupyterlite-embed
<!-- Consola Python REPL --> <iframe src="<TU-URL>/repl/index.html?kernel=python&toolbar=1" width="100%" height="500px" frameborder="0" allowfullscreen ></iframe>

⬆️ Aquí aparecerá la consola Python una vez despliegues tu JupyterLite

📓 Opción 2: Notebook Específico

Abre directamente el notebook de bienvenida. Ideal para tutoriales o demos específicas.

<!-- Notebook Específico --> <iframe src="<TU-URL>/lab/index.html?path=bienvenida-interactiva.ipynb" width="100%" height="700px" frameborder="0" allowfullscreen ></iframe>

⬆️ Aquí se abrirá el notebook interactivo con todos los ejemplos

🧪 Opción 3: JupyterLab Completo

La experiencia completa de JupyterLab en el navegador. Los usuarios pueden crear notebooks, explorar archivos, y más. Mejor para experiencias inmersivas.

<!-- JupyterLab Completo --> <iframe src="<TU-URL>/lab/index.html" width="100%" height="800px" frameborder="0" allowfullscreen ></iframe>

⬆️ Aquí aparecerá la interfaz completa de JupyterLab

📒 Opción 4: Interfaz Notebook Clásica

La interfaz clásica de Jupyter Notebook. Más simple que JupyterLab, enfocada solo en el notebook.

<!-- Jupyter Notebook Clásico --> <iframe src="<TU-URL>/notebooks/index.html?path=bienvenida-interactiva.ipynb" width="100%" height="700px" frameborder="0" allowfullscreen ></iframe>

⬆️ Aquí se mostrará la interfaz clásica del notebook

⚙️ Parámetros URL Personalizables

Agrega estos parámetros a la URL para personalizar el comportamiento:

# Abrir notebook específico ?path=nombre-notebook.ipynb # Seleccionar kernel automáticamente ?kernel=python # Mostrar/ocultar toolbar ?toolbar=1 (mostrar) ?toolbar=0 (ocultar) # Cambiar tema ?theme=JupyterLab%20Dark ?theme=JupyterLab%20Light # Combinar múltiples parámetros ?kernel=python&toolbar=1&theme=JupyterLab%20Light

Ejemplo completo:

<iframe src="<TU-URL>/repl/index.html?kernel=python&toolbar=1&theme=JupyterLab%20Light" width="100%" height="600px" frameborder="0" ></iframe>
